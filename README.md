### Backend

```bash
cd backend

python -m venv venv
source venv/bin/activate

# Run server (http://localhost:8000)
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend

npm install

# Run dev server (http://localhost:3000)
npm run dev
```
